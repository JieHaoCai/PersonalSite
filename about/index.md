#### 淘友小栈

该网站是一个虚拟游戏商品交易平台



淘游小栈（客户端和管理端），项目从零到一独立完成，目前已商用



##### 链接地址

```
http://h5.trader.cdk.vip/static-u/index.html?shopid=20
```

##### 技术栈

```
vue2+uniapp+uview
```

##### 技术亮点

```
1、商品生成自定义分享海报，可保存转发分享

2、项目接入微信H5支付和微信JSAPI支付

3、针对iOS的safari浏览器添加到桌面做不同显示效果

4、实现图片直传阿里云OSS,图片懒加载

5、微信环境静默授权登录，h5环境验证码登录
```
